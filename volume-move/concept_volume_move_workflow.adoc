= 卷移动工作流
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以在 ONTAP 9.7 或更早版本中使用 ONTAP System Manager _classic_ 界面移动卷。

在移动卷之前，您应选择卷移动操作的方法并计划此操作的时间。移动后，您可能需要更新 NDMP 备份配置。
