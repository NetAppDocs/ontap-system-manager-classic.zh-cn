= 配置对现有 SVM 的 NFS 访问
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
为 NFS 客户端添加对现有 SVM 的访问权限涉及到向 SVM 添加 NFS 配置，打开 SVM 根卷的导出策略，可选择配置 LDAP 以及从 UNIX 管理主机验证 NFS 访问。然后，您可以配置 NFS 客户端访问。
