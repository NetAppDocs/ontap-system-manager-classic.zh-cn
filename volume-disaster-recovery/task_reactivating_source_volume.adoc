= 重新激活源卷
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
当源卷变为可用时，您必须将目标卷中的数据重新同步到源卷，在重新同步操作后更新任何修改并激活源卷。
