= 配置对等关系（ ONTAP 9.2 及更早版本）
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
在 ONTAP 9.2 或更早版本的 ONTAP 9 中使用 ONTAP System Manager _classic_ 界面，您可以创建 SVM 对等关系。

对等关系用于定义网络连接，使集群和 SVM 能够安全地交换数据。您必须先创建集群对等关系，然后才能创建 SVM 对等关系。
