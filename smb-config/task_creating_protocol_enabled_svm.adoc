= 创建启用了CIFS的新SVM
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
设置启用了 CIFS 的新 SVM 涉及使用 CIFS 卷和共享创建新 SVM ，在 DNS 服务器上添加映射以及从 Windows 管理主机验证 CIFS 访问。然后、您可以配置CIFS客户端访问。
