= 使用 System Manager 启动 SVM - ONTAP 9.7 及更早版本
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以使用 ONTAP System Manager classic （适用于 ONTAP 9.7 及更早版本）通过启动 SVM 从 Storage Virtual Machine （ SVM ）提供数据访问。

.步骤
. 单击 * 存储 * > * SVMs * 。
. 选择要启动的 SVM ，然后单击 * 启动 * 。


.结果
SVM 将开始向客户端提供数据。
