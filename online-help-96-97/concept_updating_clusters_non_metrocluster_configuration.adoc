= 使用 System Manager - ONTAP 9.7 及更早版本更新非 MetroCluster 配置中的集群
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以使用 ONTAP System Manager classic （适用于 ONTAP 9.7 及更早版本）更新集群或高可用性（ HA ）对中的各个节点。要执行更新，您应选择一个 ONTAP 映像，验证集群或 HA 对中的各个节点是否已做好更新准备，然后执行更新。
